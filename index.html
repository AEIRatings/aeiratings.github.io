How to update rankings

  1. Edit the appropriate CSV in /data/ (e.g. nba.csv).
  2. CSV must include a header row with at least Team,Elo.
  3. Commit & push to GitHub — GitHub Pages will serve the updated file.

Tip: keep Elo values numeric (e.g. 1523). Optional columns (Wins, Losses) will appear if provided.
